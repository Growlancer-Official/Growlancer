# Growlancer Code of Conduct

## Our Standards

Everyone should:

- Be respectful
- Be professional
- Welcome constructive feedback
- Help others

---

## Unacceptable Behavior

- Harassment
- Hate speech
- Spam
- Plagiarism
- Sharing confidential code
- Personal attacks

---

## Enforcement

The Growlancer team reserves the right to remove contributions or revoke repository access if these guidelines are violated.
